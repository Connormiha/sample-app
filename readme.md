### Установка зависимостей
```
npm install
```

### Запуск dev окружения
```
npm start
```

### Запуск тестов
```
npm test
```

### debug тестов
```
npm run test:debug
```

### Обновить снепшоты тестов
```
npm run test:update-snapshots
```

### Покрытие тестов
```
npm run coverage
```

### Code style
```
npm run lint
```

### Storybook
```
npm run storybook
```

### Storybook build
```
npm run build-storybook
```

### Анализ сборки
```
npm run webpack:analize
```
